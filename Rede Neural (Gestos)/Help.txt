Para a instalação dos requerimentos da Rede Neural, execute os seguintes comandos no Linux:

sudo apt-get update

sudo apt-get install python3 python3-pip python3-scipy python3-dev python3-h5py python3-pillow python3-pandas

sudo apt-get install libblas-dev liblapack-dev

pip3install tensorflow==1.13.1 keras==1.2.1 sklearn


Após instalar os requerimentos, cria uma pasta denominada "videosTeste" junto ao main.py e coloque os vídeos que deseja reconhecer na pasta.

Feito isso, basta executar main.py
